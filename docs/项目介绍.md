# Memo - 长期待办事项管理应用

## 项目概述

Memo 是一个轻量级的待办事项管理应用，专注于帮助用户记录和跟踪长期任务和事件。该应用采用 Flutter 框架开发，使用 Riverpod 进行状态管理，支持本地数据存储和 WebDAV 云端同步功能。

## 核心功能

### 1. 待办事项管理
- **创建待办事项**：支持添加标题和设置截止日期时间
- **编辑待办事项**：可修改已有待办的标题和截止时间
- **完成状态切换**：标记待办事项为已完成或未完成
- **删除待办事项**：移除不需要的待办
- **列表展示**：清晰展示所有待办事项及其截止时间

### 2. 倒计时任务管理
- **长期任务跟踪**：支持长达365天的倒计时任务
- **可视化进度**：通过进度条直观展示任务完成情况
- **重复任务**：支持设置重复执行的任务
- **灵活时间设置**：可设置年、月、天为单位的持续时间
- **任务状态管理**：支持开始、暂停、完成和重置任务

### 3. 数据同步功能
- **WebDAV 协议支持**：兼容多种支持 WebDAV 的云存储服务
- **双向同步**：支持数据上传和下载
- **安全存储**：用户凭据安全存储在本地
- **自动刷新**：同步后自动刷新界面数据

### 4. 用户体验优化
- **主题切换**：支持浅色和深色主题
- **直观界面**：简洁明了的用户界面设计
- **便捷操作**：浮动按钮快速添加新任务
- **响应式设计**：适配不同屏幕尺寸

## 技术架构

### 前端框架
- **Flutter 3.27.3**：跨平台移动应用开发框架
- **Dart 3.6.1**：编程语言

### 状态管理
- **Riverpod 2.6.1**：响应式状态管理和依赖注入框架
- **StateNotifierProvider**：用于管理可变状态

### 数据存储
- **SQLite (sqflite 2.4.1)**：本地数据库存储
- **SharedPreferences 2.5.2**：用户偏好设置存储

### 网络与同步
- **WebDAV 客户端**：自定义 WebDAV 协议实现
- **Git 集成**：WebDAV 客户端通过 Git 仓库集成

### 其他依赖
- **UUID 4.5.1**：生成唯一标识符
- **Intl 0.20.2**：国际化日期格式化

## 数据模型

### 待办事项模型 (Todo)
```dart
class Todo extends TaskModel {
  String id;          // 唯一标识符
  String title;       // 待办标题
  DateTime deadline;  // 截止日期时间
  bool isCompleted;   // 完成状态
}
```

### 倒计时任务模型 (Countdown)
```dart
class Countdown extends TaskModel {
  String id;           // 唯一标识符
  String title;        // 任务标题
  DateTime startTime;  // 开始时间
  Sustain duration;    // 持续时间（年、月、天）
  bool isRecurring;    // 是否重复
  bool isCompleted;    // 完成状态
}
```

## 应用结构

### 屏幕导航
应用采用底部导航栏设计，包含三个主要页面：
1. **待办列表页面**：管理所有待办事项
2. **倒计时列表页面**：管理长期倒计时任务
3. **设置页面**：配置应用偏好和同步设置

### 状态管理架构
- **TodoListNotifier**：管理待办事项状态
- **CountdownNotifier**：管理倒计时任务状态
- **ThemeProvider**：管理应用主题状态

### 数据持久化
- 本地 SQLite 数据库存储所有任务数据
- SharedPreferences 存储用户设置和 WebDAV 凭据
- WebDAV 云端备份确保数据安全

## 使用场景

Memo 特别适合以下使用场景：

1. **长期项目管理**：跟踪持续数周或数月的项目里程碑
2. **个人目标追踪**：设定并监控长期个人发展目标
3. **重要事件提醒**：记录重要日期和截止时间
4. **习惯养成**：通过重复倒计时功能培养良好习惯
5. **工作计划**：管理工作任务和项目进度

## 项目特点

1. **专注长期任务**：不同于一般待办应用，Memo 专为需要长期跟踪的任务设计
2. **简洁高效**：界面简洁，操作直观，不增加用户认知负担
3. **数据安全**：本地存储与云端备份双重保障
4. **跨平台**：基于 Flutter，支持 Android 和 iOS 平台
5. **离线优先**：核心功能无需网络连接，仅在同步时需要网络
